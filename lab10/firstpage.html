TTML

TTML stands for Timed Text Markup Language. It was previously known as DFXP which is Distribution Format Exchange Profile. This web standard is how subtitles and captions are created with code. It is XML based and orgainizes information like XML does. TTML does have tags that are simular to HTML and it has tags that are nested in each other.

There have been several reversions of TTML. The file extension for TTML can be (.ttml) or (.xml). It is better to have TTML in a seperate file from a video. This is because when the file is seperate the text can be moved around the video to prevent from blocking certain things in the video. If they are not seperate files the subtitles can not be turned off either.

TTML is a very flexible and easier way to create subtitles. This web standard has what you need to create the subtitles. You just have to learn the tags and how each tag works. Web standards are there to show you and guide you on how it should be created. This web standard defintely does that.
